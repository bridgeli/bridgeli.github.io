# BridgeLi's Blog

欢迎来到我的个人博客源代码仓库！👋

本仓库托管于 [GitHub Pages](https://bridgeli.github.io)，使用 **Hugo** 静态网站生成器构建。这里记录了我的技术心得、生活点滴以及学习历程。

## 🌐 在线访问

请访问我的博客主页：[https://bridgeli.cn](https://bridgeli.cn)

## 🛠️ 技术栈

- **生成器**: [Hugo](https://gohugo.io/) (Extremely fast and flexible static site generator)
- **托管服务**: GitHub Pages
- **主题**: [PaperMod](https://github.com/adityatelange/hugo-PaperMod/)
- **部署**: GitHub Actions (自动构建与部署)

## 🚀 本地运行

如果您想在本地预览或开发此博客，请确保已安装 [Hugo](https://gohugo.io/getting-started/installing/) 和 [Git](https://git-scm.com/)。

1. **克隆仓库**
   ```bash
   git clone --recurse-submodules https://github.com/bridgeli/bridgeli.github.io.git
   cd bridgeli.github.io
   ```
2. **启动本地服务器**
   ```bash
   hugo server -D
   ```
3. **预览**
   打开浏览器访问 http://localhost:1313

## 📝 撰写新文章

   ```bash
   hugo new posts/my-new-post.md
   ```
